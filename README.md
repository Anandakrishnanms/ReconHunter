# 🛰️ ReconHunter 2.1  
🔍 Advanced Reconnaissance Toolkit for fast and automated target analysis.

---

## 📚 Description  

**ReconHunter 2.1** is a powerful Bash-based reconnaissance toolkit designed for penetration testers and bug bounty hunters. It automates the process of scanning targets using multiple industry-standard tools like **Masscan**, **Nmap**, **httpx**, and **WhatWeb**.

With built-in dependency management and a clean terminal UI, ReconHunter simplifies the recon workflow into a single command.

---

## 🚀 Features  

- ⚡ **Fast Port Scanning:** Uses Masscan for high-speed scanning  
- 🔎 **Service Detection:** Nmap integration for version detection  
- 🌐 **Live Web Detection:** httpx for identifying active web services  
- 🧠 **Technology Fingerprinting:** WhatWeb integration  
- 🔁 **Automatic Fallback:** Switches to Nmap if Masscan fails  
- 📦 **Automated Dependency Installation**  
- 🎨 **Stylish CLI Interface** with color output  
- 🧹 **Duplicate Removal** for clean results  

---

## 🛠️ Dependencies  

Handled automatically by the script:

- Masscan  
- Nmap  
- httpx  
- WhatWeb  
- Go (for httpx)
